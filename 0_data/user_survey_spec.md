## 1. STEP별 유저 입력 변수 리스트 (23개 전항목)

### STEP1. 기본 정보 (Demographics)
1. 성별: 남성 / 여성 / 응답하지 않음
2. 남성 전용 - 고민 영역 (복수선택): 탈모·두피 관리 / 전립선 건강 / 근육량 증가
3. 여성 전용 - 생애주기 상태: 해당없음 / 임신 준비 중 / 임신 중 / 수유 중 / 폐경기
4. 연령대: 영유아 및 어린이(만 1세~12세) / 청소년(13세~19세) / 20세~25세 / 26세~29세 / 30대 / 40대 / 50대 / 60대 이상
5. 신체 프로필: 키(cm) / 몸무게(kg) -> BMI 자동 계산 연동

### STEP2. 라이프스타일 & 일상 습관 (Lifestyle)
6. 운동 종류 및 목적 (복수선택): 안 함·체력유지재활 / 고강도 유산소 / 저항성·근력 운동 / 유연성·코어 / 고강도 인터벌
7. 음주 빈도: 전혀 안 함 / 보통 / 잦은 음주
8. 하루 카페인 섭취: 0잔 / 1~2잔 / 3잔 / 4잔 이상
9. 식습관: 일반식·불규칙 / 육식 위주 / 채식·간헐적 단식 / 완벽한 비건
10. 수면 시간: 5시간 미만 / 5~7시간 / 8시간 이상
11. 스트레스 자가인지: 1단계 / 2단계 / 3단계 / 4단계 / 5단계

### STEP3. 건강 상태 & 안전성 필터 (Medical & Safety) ★필수 민감정보
12. 흡연 여부: 비흡연 / 흡연
13. 알레르기 원료 (복수선택): 갑각류 / 대두 / 글루텐 / 유제품 / 견과류 / 어류 / 과일류 / 없음 / 기타(직접입력)
14. 과거 부작용 경험 성분 (복수선택/직접입력): 철분 / 오메가3 / 비타민C / 유산균 / 없음 / 기타 직접입력
15. 지병 및 복용 약물 (복수선택): 고혈압 / 당뇨 / 이상지질혈증 / 만성 위장질환 / 혈전 관련질환-항응고제 / 간·신장질환 / 없음(단독 선택) / 기타(직접입력)

### STEP4. 건강 고민 및 목표 (Health Goals)
16. 건강 고민 및 목표 (최대 3개 선택): 만성피로 / 눈 건조·피로 / 장 건강 / 피부탄력·이너뷰티 / 체지방감소·다이어트 / 면역력저하 / 관절보호 / 수면부족·스트레스케어 / 항노화·항산화 / 생리불순·생리통

### STEP5. 섭취 편의성 및 구매 성향 (Preference)
17. 알약 불편감: 상관없음 / 매우 불편함
18. 대안 제형 선호: 소형 알약 / 구미·젤리 / 액상·드링크 / 분말·포
19. 선호하는 영양제 형태: 정제(알약/정) / 캡슐 / 젤리 / 구미 / 액상 / 분말
20. 구매 시 우선 가치 (최대 2개 선택): 성분 함량 / 원산지·브랜드 / 첨가물 최소화 / 복용 편의성
21. 복용 환경 선호: 거치형·대용량 / 휴대형
22. 월 예산대: 1~3만원 / 3~5만원 / 5~10만원 / 10만원 이상
23. 구매 방식 선호: 정기구독 / 1회구매 / 상관없음
